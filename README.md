# UMD CS BS/MS Chatbot

A lightweight Python chatbot designed to help UMD Computer Science students better understand the Combined BS/MS application timeline, eligibility requirements, and course planning process.

Built and maintained by CS Peer Advisors.

## Features
* GPA and eligibility checks
* Graduation timeline planning
* BS/MS double-count course validation
* Study abroad considerations
* Personalized planning summary

## Tech

* Python
* Replit

## Notes

This tool is intended as a planning guideline only
